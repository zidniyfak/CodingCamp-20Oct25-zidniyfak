MounTrek Application

2024 Fullstack Development

Project Overview

MounTrek adalah aplikasi berbasis web yang dirancang untuk membantu pendaki gunung mengelola rencana perjalanan mereka. Aplikasi ini terinspirasi oleh Linktree namun dikhususkan dengan fitur manajemen konten yang lebih dalam. Pengguna dapat mendaftar, membuat halaman profil yang dipersonalisasi, dan mengelola tautan penting mereka melalui panel admin yang intuitif.


Tantangan utama dalam proyek ini adalah merancang skema database yang efisien menggunakan MongoDB untuk menangani relasi antara user dan konten link mereka secara real-time.

Key Features

  • Autentikasi Pengguna (Login/Register) yang aman.
  • Dashboard Admin Real-time.
  • Kustomisasi tema halaman profil.
  • Analitik dasar pengunjung tautan.
  • Desain Responsif untuk Mobile dan Desktop.

Technology Used

Node.js Express Tailwind CSS
